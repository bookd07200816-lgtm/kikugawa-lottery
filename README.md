# 菊川展示場 入居者感謝祭 ガラガラ抽選アプリ

セキスイハイム東海 菊川展示場の入居者感謝祭で使う、スマホ・タブレット・PC対応のWebルーレットアプリです。

## 公開URL

https://bookd07200816-lgtm.github.io/kikugawa-lottery/

## 機能

- ルーレット停止位置と当選結果を一致
- 景品残数管理
- 在庫切れ景品の自動除外
- 抽選履歴保存
- 全体リセット、履歴のみ削除
- 参加賞による不足分吸収
- 特賞、1等、2等から3等、4等以下で演出切り替え
- Web Audioによる回転音、当選音、段階別ファンファーレ
- Web Speech APIによる日本語読み上げ
- iOS Safari向けのユーザー操作による音声有効化
- 全画面表示

## 使い方

1. iPhone、iPad、PCのブラウザで公開URLを開きます。
2. 最初に「音声を有効化」を押します。
3. 「抽選スタート」を押します。
4. 当選結果、残数、履歴が自動で更新されます。
5. イベント開始前に「リセット」で初期状態に戻します。

データはブラウザの localStorage に保存されます。同じ端末・同じブラウザで続きから使えます。
